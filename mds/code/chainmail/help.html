ChainMail

Help

Every Minima user has a chainmail address, like an email address.

Users of chainmail can add contacts and send on-chain messages to other chainmail users.

All messages are signed and end-to-end encrypted so the sender, recipient and content are hidden.

Each message costs 0.001 Minima

When using ChainMail in READ mode, you must approve transactions before they will be sent.

Technical Overview

A user's chain mail address is an RSA public key.

A 'chainmail' message is a transaction sent to the chainmail address : 0x434841494E4D41494C

The transaction includes an encrypted data packet that ONLY the recipient can decrypt.. no details about the message, recipient or sender are public.

All chainmail Users try to decrypt the message.. but only the intended recipient succeeds.

The decrypted message includes the sender public key, a message and a signature.

Simple and secure.
