MiniWEB

Help

MiniWEB allows you to browse files published over MiniFS .. we call these MiniSites!

MiniSites can link to each other - just like the regular Internet

You can jump to a new site or load data from other sites - for images, text, raw data etc..

They also have access to the MDS system - in READ mode only.

This allows the more complicated sites to act as full MiniDAPPs / Applications

You can bookmark your favourite sites and they will be updated in the background when a new version is published

If you would like to create a simple site just jump straight to the Personal Site Generator from the top menu

It's very simple and easy and designed so that anyone can do it!

If you are a little more advanced and understand HTML / JS / CSS then you can build a complete website from scratch.


Technical

This section is aimed at developers who wish to get the most out of the MiniWEB system

MiniWEB sites have access to a shared folder that contains useful reusable libs, icons, images and fonts.

This way, for instance, you can include mds.js without needing to add it to your minisite and take up precious space - remember that each MiniFS file is 50k size max.

Each minisite can of course be made up of multiple MiniFS files..

Currently you have access to :

miniweb.jsThe MiniWEB lib that allows page navigation
mds.jsThe standard mds.js lib for accessing MDS functions
decimal.jsFull precision JS mathematics lib
Manrope-Regular.ttfThe default Minima font - Manrope
JQueryYou have access to the full 3.7.1 JQuery library
JQuery-UIJQuery-UI library which also has a nice icon set
FontsA folder of varied fonts
IconsA collection of SVG icons
ImagesA folder of generic wallpaper images

You can browse the files to see exactly what is available - in the ./root/miniweb/ folder of this MiniDAPP.

You can pick and choose the libs and images you want.

From the root page of your minisite you would access them like so :

<script type="text/javascript" src="../../miniweb/libs/miniweb.js"></script>
<script type="text/javascript" src="../../miniweb/libs/mds.js"></script>
<script type="text/javascript" src="../../miniweb/libs/decimal.js"></script>
And the JQuery modules are :

<link rel="stylesheet" href="../../miniweb/libs/jquery/jquery-ui-1.14.1/jquery-ui.css">
<link rel="stylesheet" href="../../miniweb/libs/jquery/jquery-ui-1.14.1/jquery-ui.icon-font.css">
<script type="text/javascript" src="../../miniweb/libs/jquery/jquery-3.7.1.min.js"></script>
<script type="text/javascript" src="../../miniweb/libs/jquery/jquery-ui-1.14.1/jquery-ui.js"></script>
And the Manrope font can be found at :

../../miniweb/fonts/Manrope-Regular.ttf

The MiniWEB functions that you have access to are..

/* Initialise the miniweb system - do this on EVERY page so your site functions correctly */
miniweb_Init();

/* Allows you to jump to a miniweb site - the MiniDAPP UID is automagically added */
miniweb_JumpToURL(minifile,callback);

/* Get the raw Byte data of the file as a Data URI */
miniweb_GetDataURI(minifile,callback);
